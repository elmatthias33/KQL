# Read Me

## Overview

The purpose of this repository is to provide supplementary training or information for the **Kusto Query Language**. 

**KQL** provides you the ability to create, store, and run interactive analysis on collected data. Sentinel uses KQL to visualize and analyze the important data, You can use KQL to create complex analytical queries that include some of the following operators: 

- Calculated columns 

- Join functions 

- Group by aggregates 

Write and run basic queries 

A query is a ***read-only request that processes data and returns the results of this processing without modifying the data or metadata***. When you construct a query, you start with either a table name or a search command.
